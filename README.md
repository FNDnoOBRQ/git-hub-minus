# 📂 Git-Hub-Mini

**Git-Hub-Mini** is a simple web application that give you ability to manage all files in a folder.  

This project is built with "Express" and "Node.js" functional for file management and a small amount of basic web-page elements and styles.  

> The style of this application is inspired by the famous **GitHub**, but of course, it’s only a simple, cheap copy.  

## ⚡ Features
- Browse and manage files in any folder
- Clean and intuitive interface
- Built with **Express** and **Node.js** for fast performance
- Simple GitHub-like UI for quick familiarity

## 🛠 Technology Stack
- **Node.js**
- **Express.js**
- **HTML / CSS / JavaScript** for frontend styling

## 📌 Note
This project is mainly for learning and experimentation purposes — it is **not a full replacement for GitHub**.
